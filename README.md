# Signova AI — Marketing Frontend

Public mirror of the Signova AI marketing frontend for design collaboration.

## Setup

```bash
npm install
npm run dev
```

## Contributing

1. Fork this repo
2. Create a feature branch: `git checkout -b feature/your-change`
3. Make your changes
4. Open a Pull Request

> **Note:** This repo contains ONLY the frontend client code. No API keys, secrets, or backend code are included.
